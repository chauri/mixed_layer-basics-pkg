• Introducion
    The model described in this document is a fully-coupled physical, biogeochemical, and ecosystem model. The generic model framework consists of a physical water column model, within which I run a biological module that adds in all the biologically-relevant state variables to this physical model.     • Model framework and equations
      The mixed-layer model simulates the evolution of water column properties under specified forcing by wind, heat, and salinity forcing. Allowance is also made for currents via a depth-independent pressure acceleration.

        The Princeton Ocean Model Formulation from of [#mellor2004user;] is used to calculate the bottom stress:

        \begin{equation}
        \tau_{x,b}=\rho C_{bot}\left(U^{2}+V^{2}\right)^{1/2}U
        \end{equation}


        \begin{equation}
        \tau_{y,b}=\rho C_{bot}\left(U^{2}+V^{2}\right)^{1/2}V
